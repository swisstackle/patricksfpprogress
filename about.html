About Patrick

Patrick is tracking key athletic metrics related to Functional Patterns training. This site documents progress over time for reference and motivation.

How this site works

Measurements are stored in data.csv. Each row contains an `exercise`, `ts` (date), `value`, and optional `youtubeId` for the latest video. Charts are rendered client-side with Chart.js.
